Proyectos

Sistema Web para Administración de Eventos Culturales y Deportivos
El sistema se encarga de registrar a los usuarios que realizan publicaciones de eventos de su recinto, estos eventos se realizan en diferentes lugares del Estado de México, por lo que se agrega la api de google maps para mostrar la ubicación del lugar donde donde se realizará el evento.

El sistema se encarga de mostrar los eventos activos hasta el campo fecha, los eventos no activos no se muestran en el cartelera general del sistema, pero pueden ser consultados por el usuario que los registró.
Aplicación Móvil para Consulta de Eventos Culturales y Deportivos
La app móvil es un medio por el cual la sociedad puede realizar consultas de eventos de recintos asociados a la Secretaría de Cultura
Sistema de Ingresos Hortaliza
Este es un proyecto que realicé para la Autopista Toluca-Zitácuaro el cuál se realizó mediante MVC utilizando PHP como lenguaje de Back.
Sistema Gestor de Telepeaje
Este es un proyecto que realicé para la Autopista Toluca-Zitácuaro, el cual se encarga de auditar que cada uno de los cruces de vehículos que pasan por las casetas se cobren correctamente.
